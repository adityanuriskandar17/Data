# Kursus Data Engineering (Bahasa Indonesia & Inggris)

## Kursus Gratis

### 1. Data Engineering Zoomcamp (DataTalks.Club)
- **Level**: Pemula - Menengah
- **Durasi**: 8 minggu
- **Materi**:
  - Docker, Terraform, SQL
  - Data Warehousing (BigQuery)
  - Batch Processing (Spark)
  - Stream Processing (Kafka)
  - Pipeline Orchestration (Airflow)
  - dbt for transformations
- **Link**: https://github.com/DataTalksClub/data-engineering-zoomcamp

### 2. Google Data Analytics Certificate (Coursera)
- **Level**: Pemula
- **Durasi**: 6 bulan (10 jam/minggu)
- **Cakupan**: SQL, Spreadsheet, R, Tableau

### 3. Database Design & Programming with SQL (Oracle Academy)
- **Level**: Pemula
- **Fokus**: SQL dasar - mahir

## Kursus Berbayar

### 1. Data Engineering Specialization (Coursera - DeepLearning.AI)

- **Platform**: AWS, Spark, Airflow

### 2. Data Engineering on Google Cloud (Coursera)
- **Level**: Intermediate
- **Platform**: GCP (BigQuery, Dataflow, Dataproc)

### 3. AWS Certified Data Analytics Speciality (Udemy)
- **Level**: Advanced
- **Persiapan**: Sertifikasi AWS

### 4. Data Engineering with Databricks (Databricks Academy)
- **Level**: Intermediate
- **Fokus**: Lakehouse, Spark, Delta Lake
