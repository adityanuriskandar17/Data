# Panduan Sertifikasi Data Engineering

## Mengapa Sertifikasi?
- Validasi skill secara formal
- Meningkatkan kredibilitas di pasar kerja
- Memperluas wawasan (belajar best practices)
- Requirement di beberapa perusahaan/konsultan

## Sertifikasi Cloud

### 1. AWS Certified Data Analytics - Specialty
- **Biaya**: $300
- **Durasi**: 180 menit, 65 soal
- **Topik**:
  - Collection (Kinesis, SQS)
  - Storage (S3, DynamoDB)
  - Processing (Glue, EMR, Athena)
  - Analysis (Redshift, QuickSight)
  - Visualization
- **Masa berlaku**: 3 tahun

### 2. Google Professional Data Engineer
- **Biaya**: $200
- **Durasi**: 120 menit, case study
- **Topik**:
  - BigQuery, Dataflow, Dataproc
  - Pub/Sub, Cloud Storage
  - ML dengan TensorFlow dan AI Platform
  - Data governance dan security

### 3. Microsoft Azure Data Engineer Associate (DP-203)
- **Biaya**: $165
- **Topik**:
  - Azure Data Lake, Synapse Analytics
  - Data Factory, Databricks
  - Stream Analytics, Cosmos DB

## Strategi Belajar

1. **Pilih satu cloud provider** dan kuasai (AWS paling banyak dipakai)
2. **Mulai dari fundamental** sebelum specialty
3. **Praktik hands-on** minimal 50% dari waktu belajar
4. **Gunakan practice test** untuk familiar dengan format ujian
5. **Gabung study group** di Discord/Telegram
6. **Dokumentasi resmi** adalah sumber terbaik

## Tips Lulus

- Buat jadwal belajar 1-2 jam per hari selama 2-3 bulan
- Kerjakan mini project di cloud masing-masing
- Gunakan flashcards untuk konsep penting
- Istirahat cukup sebelum ujian
